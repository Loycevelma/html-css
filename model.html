BOX MODEL CSS

What is a box model

It is used to design layout of web pages and html tags

Main properties of a Box Model


width-It specifies on width of an element within a content.
Height-It specifies on the height of an element within a content.
Padding-THis is a space within a content.
Border-This is a space between an element border and the content.
Margin-This is a space around an element.

Calculate the total width of a paragraph given
width=400px;
Padding=20px;
Margin=20px;
Border=2px-solid


padding-Top to bottom-20*2 + Left to right-20*2=80
Margin-Top to bottom-20*2 + Left to right-20*2=80
Border-Top to bottom-2*2 + Left to right-2*2=8
Total width-80+80+8+400=568
